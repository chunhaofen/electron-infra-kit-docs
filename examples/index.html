示例 ​

本节提供了一系列实用的示例，帮助你快速理解和使用 electron-infra-kit 的各项功能。每个示例都包含完整的代码和详细的说明。

可用示例 ​

基础配置 ​

学习如何从零开始配置一个使用 electron-infra-kit 的 Electron 应用。这个示例展示了最简单的项目结构和配置方式。

适合场景：

  • 刚开始使用 electron-infra-kit
  • 需要快速搭建项目原型
  • 了解基本的初始化流程

涵盖内容：

  • 主进程初始化
  • 预加载脚本配置
  • 渲染进程基本使用

多窗口应用 ​

学习如何创建和管理多个窗口，以及如何在窗口之间进行通信。这个示例展示了 WindowManager 的强大功能。

适合场景：

  • 开发多窗口应用（如 IDE、设计工具）
  • 需要管理复杂的窗口生命周期
  • 实现窗口间的数据传递

涵盖内容：

  • 创建不同类型的窗口
  • 窗口状态管理
  • 窗口间通信
  • 窗口生命周期钩子

IPC 通信 ​

学习如何使用 IpcRouter 实现类型安全的进程间通信。这个示例展示了如何定义处理器、注册路由和在渲染进程中调用。

适合场景：

  • 需要在主进程和渲染进程之间传递数据
  • 希望实现类型安全的 API 调用
  • 需要参数验证和错误处理

涵盖内容：

  • 定义 IPC 处理器
  • 参数验证（使用 Zod）
  • 依赖注入
  • 错误处理
  • 渲染进程调用

状态同步 ​

学习如何使用 MessageBus 在多个窗口之间同步状态。这个示例展示了如何实现跨窗口的数据共享和实时更新。

适合场景：

  • 需要在多个窗口间共享状态
  • 实现实时数据同步
  • 管理全局应用状态

涵盖内容：

  • 设置和获取共享数据
  • 监听数据变化
  • 权限控制
  • 跨窗口状态同步

完整应用示例 ​

一个综合使用 electron-infra-kit 所有功能的完整应用示例。这个示例展示了如何在实际项目中组织代码和使用各个模块。

适合场景：

  • 需要一个完整的参考实现
  • 了解最佳实践和代码组织
  • 学习如何集成所有功能

涵盖内容：

  • 完整的项目结构
  • 窗口管理
  • IPC 通信
  • 状态同步
  • 配置管理
  • 错误处理
  • 调试工具

如何使用这些示例 ​

  1. 阅读代码：每个示例都包含完整的、可运行的代码
  2. 理解概念：通过注释和说明理解每个功能的用途
  3. 动手实践：将代码复制到你的项目中并进行修改
  4. 参考文档：结合 API 参考 深入了解每个功能

需要帮助？ ​

  • 查看 快速开始指南
  • 阅读 核心概念文档
  • 访问 GitHub Issues
